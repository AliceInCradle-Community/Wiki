### Control List

| Key	| Menu Operations	| Gameplay Operations |
| --- | --- | --- |
| `↑`	| Move | Cursor	| Jump |
| `←`	| Move | Cursor	| Jump |
| `→`	| Move | Cursor	| Jump |
| `↓`	| Move | Cursor	| Crouch |
| `Space`	| Confirm/Talk/Search	| Run |
| `Z`	| Confirm/Talk/Search	| Attack |
| `X`	| Cancel／Skip	| Magic |
| `C`	| -	| Target |
| `A`	| -	| Map |
| `S`	| -	| Other |
| `D`	| -	| Item |
| `Q`	| -	| Go to Left Tab |
| `E`	| -	| Go to Right Tab |
| `LShift`	| -	| Evade/Shield |
| `Esc`	| Cancel	| Menu |
| `Z`+`X`	| - | chant cancel |
| `↓`+`←`or`→`	| - | Crouch move |
| `C` long press	| - | Hispeed stop reel |
